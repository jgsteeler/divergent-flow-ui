# Divergent Flow Branding Assets

This folder contains branding assets for Divergent Flow, matching the UI light production theme.

## Files

- `divergent-flow-logo.svg`: App logo and favicon (SVG)
- `divergent-flow-banner.svg`: Horizontal banner for web/app/marketing (SVG)

## Colors Used

- Primary: #004687 (deep blue)
- Secondary: #7BB2DD (powder blue)
- Accent: #C09A5B (gold)
- Background: #F7FAFC
- Text: #1A202C

## Usage

- Use the logo SVG for favicon, app icon, and anywhere a compact logo is needed.
- Use the banner SVG for headers, marketing, and web/app branding.

Feel free to request PNG exports or other formats as needed.
